# MOCHA
MOCHA: a tool for MObility CHaracteristics Analysis

# How to use:
Usage: Principal.py ['-p', '-e', '-pe'] ['-r', '-s', '-n'] filename  
['-p', '-e', '-pe']: parse, extract, or parse and extract  
['-r', '-s', '-n' ]: RAW, SWIM or NS2 (if parsing)  
filename        : file to be parsed/processed
